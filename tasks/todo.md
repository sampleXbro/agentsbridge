## 1. Scope and canonical surface

The **only** canonical inputs under `.agentsmesh/` are these 8 feature families (`src/core/canonical-types.ts:106`):

| Family | Canonical source | Notes |
|--------|------------------|-------|
| rules | `rules/*.md` (`CanonicalRule`) | Has `root`, `targets[]`, `globs[]`, `codexEmit` |
| commands | `commands/*.md` (`CanonicalCommand`) | Has `outputStyle` flag |
| agents | `agents/*.md` (`CanonicalAgent`) | Has `outputStyle` flag |
| skills | `skills/<name>/SKILL.md` (`CanonicalSkill`) | Supporting files allowed |
| mcp | `mcp.json` (`McpConfig`) | |
| permissions | `permissions.yaml` (`Permissions`) | allow / deny / ask |
| hooks | `hooks.yaml` (`Hooks`) | |
| ignore | `ignore` (`IgnorePatterns`) | |

**Rule ŌĆö gap in scope only if:**
1. A canonical input above maps to a target output the doc describes, AND
2. The generator fails to emit it, emits it wrong, or suppresses it without doc authority.

**Target-native features without canonical source are out of scope** even if the `docs/agent-structures/*` file mentions them (they belong on a product roadmap, not in this audit). See ┬¦5.

If any of these should become canonical, extend `src/core/canonical-types.ts` and `.agentsmesh/` schema first, then re-open the gap.

## 8. Suggested fix order (independent of this audit)

1. **Define the `~/` expansion contract** ŌĆö one helper or one documented convention; then fix `cursor/continue/cline` global paths.
2. **Wire roo-code modes** ŌĆö canonical `agents[]` ŌåÆ `.roomodes` / `settings/custom_modes.yaml`. Largest single canonical-source gap.
3. **Close P0s**:
   - Remove `src/AGENTS.md` from windsurf managed files (or scope it).
   - Generate `~/.copilot/AGENTS.md` (or delete the unused constant and update the doc).
   - Suppress antigravity project MCP emission (or move to global-only).
   - Stabilise gemini-cli `mirrorGlobalPath` against constant changes.
4. **Implement cheap canonical-sourced features** ŌĆö copilot project `generateMcp`, cline `.clinerules/` managed dir.
5. **Sweep unused constants** ŌĆö `copilot` (4), `gemini-cli` (1). Delete or wire up.
6. **Doc reconciliation pass** ŌĆö ┬¦6 list above, including normalizing all target docs to `general.md` and dropping `AGENTS.override.md` references.
