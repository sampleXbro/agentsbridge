## Requirements

- **Node.js 20** or later
- Linux, macOS, or **Windows** (native, not WSL)
- Any package manager: npm, pnpm, or yarn

<Aside type="tip">
  **Windows support** is first-class: paths are normalized internally so generated configs and `installs.yaml` are portable across platforms. Watch mode uses polling on Windows because `ReadDirectoryChangesW` can miss just-created files in tmpdirs. CI runs the full test suite on Linux, macOS, and Windows.
</Aside>

## Install as a dev dependency

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install -D agentsmesh
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add -D agentsmesh
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add -D agentsmesh
    ```
  </TabItem>
</Tabs>

## Run without installing

```bash
npx agentsmesh --help
```

## CLI aliases

AgentsMesh installs two aliases:

```bash
agentsmesh --version   # full name
amsh --version         # short alias
```

Both commands are identical. Use whichever you prefer.

## Verify installation

```bash
agentsmesh --version
# 0.7.0

agentsmesh --help
# Usage: agentsmesh <command> [options]
# ...
```

<Aside type="note">
  If you installed locally (not globally), prefix commands with `npx agentsmesh` or add a script to your `package.json`.
</Aside>

## IDE autocomplete via JSON Schema

Every canonical config file ships with a generated JSON Schema, so VS Code, JetBrains, and other editors give you autocomplete and validation out of the box. `agentsmesh init` writes the appropriate `# yaml-language-server: $schema=...` directive (or `$schema` field for JSON) into each file:

| Config file | Schema location (in your `node_modules`) |
|---|---|
| `agentsmesh.yaml` / `agentsmesh.local.yaml` | `agentsmesh/schemas/agentsmesh.json` |
| `.agentsmesh/hooks.yaml` | `agentsmesh/schemas/hooks.json` |
| `.agentsmesh/permissions.yaml` | `agentsmesh/schemas/permissions.json` |
| `.agentsmesh/mcp.json` | `agentsmesh/schemas/mcp.json` |
| `.agentsmesh/packs/*/pack.json` | `agentsmesh/schemas/pack.json` |

## Add scripts to package.json

```json
{
  "scripts": {
    "sync": "agentsmesh generate",
    "sync:check": "agentsmesh check",
    "sync:watch": "agentsmesh watch"
  }
}
```
